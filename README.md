# 🏨 تطبيق مرحبتين - Moraheetain App

## نبذة عن التطبيق
تطبيق حجز الاستراحات اليومي في ليبيا - منصة متقدمة تربط أصحاب الاستراحات بالنزلاء

### الميزات الرئيسية:
- ✅ حسابات آمنة للنزلاء وأصحاب الاستراحات
- ✅ نظام حجز ذكي مع تقويم تفاعلي
- ✅ طرق دفع متعددة (أونلاين + كاش + تحويل بنكي)
- ✅ نظام تقييمات وتقيمات
- ✅ دعم كامل للغة العربية
- ✅ ذكاء اصطناعي للتوصيات

---

## 🚀 البدء السريع

### المتطلبات:
- Node.js v16+
- MongoDB
- npm أو yarn

### التثبيت:
```bash
# استنساخ المستودع
git clone https://github.com/zadigkf097-maker/moraheetain.git

# الدخول للمجلد
cd moraheetain

# تثبيت المكتبات
npm install

# تشغيل البرنامج
npm start
```
